ZTNA Web Site

Keycloak + Node.js + Nginx ile kimlik doğrulama/yetkilendirme akışı ve güvenli erişim.

Öne çıkanlar

  • OIDC login flow, session yönetimi
  • JWT doğrulama, role/permission kontrolü
  • Reverse proxy arkasında güvenli yapı

Kullandıklarım

  • Keycloak, OpenID Connect
  • Node.js (Express), jose/jwt
  • Nginx, SSL (opsiyonel)
