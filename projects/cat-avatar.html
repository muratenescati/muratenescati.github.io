Interactive Desktop Cat Avatar

Unity’de animasyon oynatan, Python tarafında komut/AI işleyen masaüstü avatar sistemi.

  • Komut dosyası / IPC ile Unity kontrolü

Kullandıklarım

  • Unity (C#), Animation Controller
  • Python, LLM entegrasyonu
